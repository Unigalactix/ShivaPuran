## Local preview

This is a pure static site. Open `index.html` directly, or run any static server:

```pwsh
# Option 1 — Python
python -m http.server 5500

# Option 2 — Node (npx)
npx serve .
```

Then visit `http://localhost:5500/`.

## Deploy to GitHub Pages

1. Create a public repo (e.g. `SHIVAPURAN`).
2. Push this folder to `main`.
3. Settings → Pages → Build from branch → `main` / `/ (root)`.
4. `.nojekyll` is already included so Pages skips Jekyll processing.

## Tech

- Plain HTML5 + CSS3 + a sprinkle of vanilla JS — **no build step**.
- Google Fonts: *Cormorant Garamond*, *Cinzel*, *Inter*, *Noto Serif Devanagari*.
- Responsive, accessible, zero dependencies, zero tracking.
